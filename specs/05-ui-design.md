# 05. 画面・UX 設計

## 5.1 デザイン方針

- **モダン・ミニマル**: shadcn/ui + Tailwind による一貫感のあるデザイン。
- **読みやすさ重視**: ダーク／ライト両モード対応、デフォルトはシステム追従。
- **片手で打刻**: スマホでもメインアクション（開始/終了）に必ず到達できるレイアウト。
- **数値はモノスペース**: 時間・金額は等幅フォントで揃える。
- **カラー**: プロジェクトごとのカラーをアクセントとして全体に反映。

## 5.2 画面構成

```
┌─────────────────────────────────────────────────┐
│ Header: ロゴ / 月切替 / アバター(ログアウト)    │
├─────────────────────────────────────────────────┤
│ Sidebar (PC) or Bottom Nav (SP):                │
│   - ダッシュボード（タイマー）                  │
│   - プロジェクト                                │
│   - 打刻一覧                                    │
│   - 月次サマリー                                │
└─────────────────────────────────────────────────┘
```

## 5.3 各画面の仕様

### 5.3.1 `/login`

- 中央に「Google でログイン」の大きなボタン 1 つ。
- 許可外メールでサインインした際は赤色のエラーバナー。
- フッターに小さく「自分用タイムカード v0.1」。

### 5.3.2 `/`（ダッシュボード ＝ タイマー画面）

レイアウト（モバイル想定）:
```
┌────────────────────────────┐
│ 今月のサマリー（コンパクト）│
│  合計: 120.5h / ¥482,000   │
├────────────────────────────┤
│ ▶ 進行中の打刻             │
│  ProjectA  00:42:11        │
│  メモ入力 textarea         │
│  [ 終了 ] (赤いボタン)     │
├────────────────────────────┤
│ ＋ 新しい打刻を開始        │
│   プロジェクト選択(チップ) │
│   メモ(任意)               │
│   [ 開始 ] (大ボタン)      │
├────────────────────────────┤
│ 直近の打刻 (最大10件)      │
│  時刻 / プロジェクト / 時間│
└────────────────────────────┘
```

PC では「進行中 + 新規開始」を左カラム、「直近の打刻」を右カラムに配置。

**インタラクション**
- 進行中エントリがある場合、新規開始ボタンは disabled で「先に終了してください」とツールチップ。
- タイマーは `useEffect` + `setInterval(1000)` でクライアント側のみ駆動。
- メモは入力後、blur で自動保存（Server Action）。

### 5.3.3 `/projects`

- カード型一覧。各カードに color のアクセント、名前、既定時給 / 今月時給、当月稼働時間。
- 右上に「新規作成」ボタン（モーダルでフォーム）。
- カード右上の「…」メニューから「編集 / アーカイブ」。
- アーカイブ済みは折りたたみで別セクション。

**プロジェクトカード wireframe**
```
┌──────────────────────────────────────────┐
│ ●ProjectName                       ⋮     │
│  既定 ¥5,000/h  今月 ¥5,500/h            │
│  今月: 24.0h / ¥132,000                  │
└──────────────────────────────────────────┘
```

### 5.3.4 `/projects/[id]`

- プロジェクト詳細（編集フォーム + 月次レート表）。
- 月次レート表: `[ 2026-05 | ¥5,500 | 編集 / 解除 ]`
- 「この月のレートを設定」モーダル。
- 当該プロジェクトの直近打刻（無限スクロール）。

### 5.3.5 `/entries`

- フィルター: 月セレクタ + プロジェクトセレクタ。
- テーブル形式（モバイルではカード形式）。

```
| 日付       | 時間帯        | プロジェクト | 稼働 | メモ抜粋 | 操作 |
|------------|---------------|-------------|------|---------|------|
| 2026-05-08 | 09:00 - 12:30 | ProjectA    | 3.5h | 設計... | 編集/削除 |
```

- 各行クリックで編集モーダル。
- 「＋ 手動で追加」ボタン → モーダル（プロジェクト, 開始, 終了, メモ）。
- 削除は確認ダイアログ。

### 5.3.6 `/summary`

- 月セレクタ（前月／次月／カレンダー風 picker）。
- カード x プロジェクト数。各カードに色アクセント + 数値。
- 上部に大きな合計（時間・金額）。
- カードクリックで `/entries?yearMonth=...&projectId=...` に遷移。

```
┌────────────────────────────────────────┐
│  2026年5月 サマリー                    │
│  ───────────────────                   │
│  合計  120.5h  /  ¥482,000             │
└────────────────────────────────────────┘
┌──────────────┐ ┌──────────────┐
│ ●ProjectA    │ │ ●ProjectB    │
│ 60.0h         │ │ 60.5h         │
│ ¥330,000      │ │ ¥152,000      │
│ 適用 ¥5,500/h │ │ 適用 ¥2,500/h │
│ (月次)        │ │ (既定)        │
└──────────────┘ └──────────────┘
```

## 5.4 共通コンポーネント

| コンポーネント | 役割 |
|----------------|------|
| `ProjectChip` | プロジェクト名 + カラードット |
| `Timer` | 経過時間表示（HH:MM:SS）、`startedAt` を props |
| `MonthSwitcher` | 前月／次月 + month picker |
| `EntryForm` | 打刻の作成・編集フォーム（モーダル内で再利用） |
| `RateBadge` | 適用レート（`既定` / `月次` バッジ付き） |
| `EmptyState` | データなし時の案内 |
| `LoadingButton` | 連打防止つきボタン |

## 5.5 アクセシビリティ

- フォーム要素は `<label>` をすべて関連付け。
- ボタンは `aria-busy` で処理中表現。
- カラーコントラストは WCAG AA 準拠（shadcn/ui のデフォルトに準拠）。
- キーボードのみで主要操作（開始 / 終了 / 編集）が可能。

## 5.6 表示フォーマット

| 種類 | 形式 | 例 |
|------|------|------|
| 日付 | `YYYY-MM-DD (ddd)`（JST） | `2026-05-08 (金)` |
| 時刻 | `HH:mm`（JST） | `09:30` |
| 期間 | `H時間M分` または `123:45` 形式 | `3時間30分` |
| 金額 | `¥` + カンマ区切り | `¥132,000` |
| 月 | `YYYY年M月` | `2026年5月` |
